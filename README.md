# 🩺 Diabetes Prediction using Machine Learning

This project aims to predict whether a person has diabetes or not using machine learning techniques. It uses a dataset containing medical attributes such as glucose level, BMI, age, and more to train a classification model.

---

## 📌 Project Overview

Diabetes is a serious health condition that can lead to complications if not detected early. This project demonstrates how machine learning can be used to assist in early diagnosis by analyzing patient health data.

The notebook includes:

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection and scaling
* Model training and evaluation

Such projects typically use datasets like the Pima Indians Diabetes dataset with features like glucose, insulin, BMI, and age. 

---

## 📊 Dataset

The dataset contains medical predictor variables and one target variable (`Outcome`).

### Features include:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

### Target:

* `Outcome` (0 = Non-diabetic, 1 = Diabetic)
